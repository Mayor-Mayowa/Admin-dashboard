# Dashboard Demo

A responsive admin dashboard built as part of The Odin Project's 
Intermediate HTML and CSS course.

## Features

- Dark themed UI with a leather orange accent
- Responsive grid layout
- Sticky sidebar navigation
- Project cards with hover effects
- Announcements and trending sections
- Custom DiceBear avatars

## Built With

- HTML
- CSS (Grid, Flexbox, Custom Properties)

## Live Demo

https://mayor-mayowa.github.io/Admin-dashboard/

## What I Learned

Getting the CSS Grid layout right was the main challenge — specifically 
making the sidebar stick while the main content scrolls independently. 
Also spent time building a consistent design system using CSS custom 
properties for colors, shadows, and typography.
